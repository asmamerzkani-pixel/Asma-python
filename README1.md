# Productivity App

Student: Asma Merzkani

## Description
This is a simple Productivity Application developed in Python using SQLite.
The application allows users to register, login, and manage their daily tasks.

## Features
- User registration and login
- Add tasks
- Show tasks
- Delete tasks
- SQLite database

## Files
- main.py : main program
- database.py : database initialization
- user.py : user management
- task.py : task management
- project.pdf : project report

## How to Run
Run the program using:
python main.py
